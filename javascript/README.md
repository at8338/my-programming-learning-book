# Javascript
Javascript (Nodejs)

適当にnodebrewなどをインストールしJavascriptの実行環境を整える.

## javascripting
セキュリティキャンプの予習のためにJavascriptを始めた.

```bash
$ npm install -g javascripting
```

## learnyounode

``` bash

```
